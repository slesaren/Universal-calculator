# Универсальный калькулятор

 Windows Forms приложение на C# (.NET 8.0), поддерживающее три режима работы:
- **p-ичные числа** (основания от 2 до 16)
- **Простые дроби** (с возможностью сокращения и преобразования в смешанные)
- **Комплексные числа** (с поддержкой операций над мнимой единицей)

Проект реализован с использованием объектно-ориентированного подхода и паттерна MVC (Model-View-Controller).

##  Возможности

### Общие функции
- Базовые арифметические операции: сложение, вычитание, умножение, деление
- Унарные операции: смена знака, возведение в квадрат, обратное число (1/x)
- Работа с памятью: MS, MR, M+, MC
- Буфер обмена: копирование, вставка, вырезание
- Поддержка горячих клавиш
- Адаптивный интерфейс с изменяемым размером шрифта

### Режим p-ичных чисел
- Поддержка систем счисления от 2 до 16
- Ввод цифр A-F для 16-ричной системы
- Десятичный разделитель (точка)
- Динамическое переключение основания с конвертацией текущего значения

### Режим дробей
- Ввод в формате `числитель/знаменатель`
- Автоматическое сокращение дробей
- Преобразование в смешанную дробь
- Выделение целой части
- Обратная дробь

### Режим комплексных чисел
- Ввод в формате `a+bi` или `a-bi`
- Операции с мнимой единицей
- Сопряжённое число
- Извлечение действительной и мнимой частей
- Вычисление модуля и аргумента (в градусах)
- Переключение редактирования между частями


## Требования

- .NET 8.0 SDK или выше
- Windows 10/11 (приложение использует Windows Forms)
- Visual Studio 2022 (рекомендуется) или любая другая IDE с поддержкой .NET

## Установка и запуск

### Сборка из исходного кода
```bash
   git clone https://github.com/slesaren/Universal-calculator
   cd stp_k
   dotnet restore
   dotnet build
   dotnet run
```

### Установка через инсталлятор
1. Запустите CalculatorSetup1.msi
2. Следуйте инструкциям мастера установки
3. После установки приложение появится в указанной папке
